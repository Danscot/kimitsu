Le bot

Rencontre Kimitsu.

Kimitsu est un mini bot WhatsApp conçu pour simplifier ta vie dans les groupes et les conversations. Médias, tags, téléchargements, réponses automatiques — tout se contrôle avec de simples commandes.

Connecte ton numéro une seule fois, et le bot s'active automatiquement. Pas de configuration complexe, pas d'application à installer.

Processus

Prêt en trois étapes.

01
Crée ton compte

Inscris-toi sur bot.danscot.dev en quelques secondes. Aucune carte bancaire pour l'essai gratuit.

02
Connecte ton numéro

Suis le tutoriel de connexion : entre ton numéro WhatsApp et entre le code reçu pour lier le bot.

03
Le bot s'active

Une fois connecté, ton numéro a le bot activé automatiquement. Tape une commande et Kimitsu répond.

Fonctionnalités

Les commandes principales.

Médias
/vvContourner les vues uniques
/saveSauvegarder les médias dans tes MP, même les vues uniques
/stickerCréer des stickers depuis une image ou vidéo
Tags
/tagMentionner tous les membres sur un message spécifique
/tagallTagger tous les utilisateurs du groupe
/tagadminTagger uniquement les administrateurs
Téléchargeur
/playTélécharger de l'audio depuis YouTube
/tiktokTélécharger des vidéos TikTok sans filigrane
Configuration
/setprefixModifier le préfixe des commandes
/antidelActiver ou désactiver la récupération des messages supprimés
/settagDéfinir le message automatique lors des tags
/publicActiver ou désactiver le mode public
/respons onActiver la réponse automatique définie par /settag
/respons offDésactiver la réponse automatique
/statusVoir l'état du bot et le temps de fonctionnement
